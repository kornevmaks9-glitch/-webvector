# project_specs.md

## Что делает приложение и кто им пользуется
Сайт-визитка компании **WebVector** — мы делаем и продаём сайты для бизнеса по всей России.
Аудитория: владельцы бизнеса, которым нужен сайт (лендинг, интернет-магазин, корпоративный сайт).
Цель сайта — показать услуги и цены, вызвать доверие и подвести человека к заявке/контакту.

### SEO-фокус (главный приоритет)
Главный город-плацдарм — **Смоленск**. Цель — топ-1 в Яндексе по запросу
«создание сайтов под ключ Смоленск» и близким геозапросам. Главная страница — это
«документ» под этот запрос: ключ в H1/title/description, упоминание города в тексте,
геопривязка в JSON-LD. Остальные 14 регионов (Москва, СПб и т.д.) — зона работы для
расширения позже, перечислены в блоке «География работы». Конкурент — фрилансер,
ранжирующийся по «создание сайтов под ключ Смоленск».

Дизайн копирует структуру и стиль референса (тёмная тема, огромная типографика, плавные анимации,
крупная 3D-подобная картинка в hero), но контент — про веб-разработку, а не NFT.

## Технический стек
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Деплой: Vercel
- **Без Supabase и без базы данных** — на сайте нет авторизации и нет данных, которые нужно сохранять.
  Весь контент (услуги, цены, текст блога) живёт прямо в коде. Это отступление от стандартного стека
  в CLAUDE.md, потому что для сайта-визитки с одним блог-постом БД не нужна — добавлять её было бы
  лишней сложностью.
- Форма заявки — без backend: либо `mailto:`, либо просто визуально оформленная форма с алертом
  "заявка отправлена" (без реальной отправки, т.к. нет почтового сервиса). Можно подключить
  Telegram-бота или Resend позже, если потребуется.

## Страницы и пользовательские сценарии

### `/` — главная
- Шапка: лого "WebVector", меню (Услуги, О нас, Блог, Контакты), кнопка "Оставить заявку"
- Hero: крупный заголовок про создание сайтов, подзаголовок, абстрактная анимированная 3D-графика
  (CSS/SVG, без внешних 3D-моделей), кнопка "Оставить заявку"
- Услуги (3 карточки вместо NFT-артворков):
  1. Лендинг — от 30 000 ₽
  2. Интернет-магазин — от 80 000 ₽
  3. Корпоративный сайт — от 60 000 ₽
- О нас — короткий блок про компанию
- Блог — превью трёх свежих постов со ссылкой на список статей `/blog`
- Контакты / футер — Telegram, MAX, телефон, email (плейсхолдеры, заменишь на свои), форма заявки

### `/blog` — список статей
Лента всех постов (дата, заголовок, описание) со ссылками на статьи. Точка входа в блог из меню.

### `/blog/[slug]` — статья
Динамический маршрут: одна страница рендерит любой пост по slug. Все посты лежат массивом
`BLOG_POSTS` в `lib/content.ts` (заголовок, описание, дата, ключевые слова, тело статьи блоками),
страницы генерируются статически (`generateStaticParams`). Каждая статья оптимизирована под
SEO-запросы про покупку и заказ сайтов (ключевые слова в `keywords`, Article JSON-LD, OG-картинка).

## Модели данных
Нет базы данных. Все тексты (услуги, цены, посты блога) хранятся как константы в `lib/content.ts`.
Тело каждого поста — массив блоков (`h2` / `p` / `ul` / `prices`), который рендерит `/blog/[slug]`.

## Сторонние сервисы
Нет (Telegram/MAX — просто внешние ссылки-плейсхолдеры, без интеграции API).

## Что значит «готово»
- `npm run build` проходит без ошибок
- `npm run dev` запускается, главная и страница блога открываются без ошибок в консоли
- Вёрстка адаптивна (мобильный/десктоп)
- Визуально передан стиль референса: тёмный фон, крупный шрифт, аккуратные анимации
- Кнопки контактов — рабочие плейсхолдер-ссылки, которые легко поменять на реальные
